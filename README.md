# 🌍 Europe Weather Explorer

A Flask web application that displays information about European countries and live weather data for their capitals.

## Features
- Browse all European countries via a dropdown menu
- View country flag, capital city and population
- See real-time weather data for the selected capital (temperature, humidity, wind speed, UV index)

## APIs Used
- [REST Countries](https://restcountries.com/v3.1/region/europe) — country data
- [WeatherAPI](https://www.weatherapi.com/) — live weather data

## Tech Stack
- Python 3.12
- Flask 3.x
- Requests library
- HTML5 / CSS3 / JavaScript

## How to Run
1. Clone the repository
2. Install dependencies:
   pip install flask requests
3. Add your WeatherAPI key in app.py
4. Run the app:
   python app.py
5. Open http
